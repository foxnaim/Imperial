## About

Imperial is a JavaScript web application built with React and Vite. Features a modern architecture with fast development experience.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React + Vite |
| Language | JavaScript |
| Build | Vite |

## Getting Started

```bash
git clone https://github.com/foxnaim/Imperial.git
cd Imperial
yarn install
yarn dev
```
